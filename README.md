Discord Auto-Status Changer (Self-bot)
이 프로젝트는 디스코드 계정의 상태(온라인/자리비움 등)와 활동 메시지를 일정 간격으로 무작위 변경해주는 파이썬 셀프봇 스크립트입니다.

⚠️ 경고 (매우 중요)
ToS 위반: 디스코드는 일반 계정을 이용한 자동화(셀프봇)를 서비스 약관(ToS)으로 엄격히 금지하고 있습니다.

정지 위험: 이 스크립트를 사용하여 발생하는 모든 계정 정지 등의 불이익은 사용자 본인에게 있습니다. 반드시 테스트용 계정에서만 사용하십시오.

API 제한: 너무 짧은 간격으로 상태를 변경할 경우 디스코드 서버로부터 차단(Rate Limit)을 받을 수 있습니다.

🚀 기능
무작위 변경: 상태 아이콘(온라인, 자리비움, 방해금지, 오프라인)과 활동 메시지를 리스트에서 무작위로 선택합니다.

중복 방지: 직전에 사용된 아이콘이나 메시지가 다시 선택되지 않도록 로직이 적용되어 있습니다.

사용자 정의: STATUS_TYPES 및 STATUS_LIST 배열을 수정하여 원하는 상태로 쉽게 변경 가능합니다.

🛠 사용 방법
1. 환경 설정
discord.py-self 라이브러리가 필요합니다.

Bash


pip install discord.py-self
2. 토큰 설정
디스코드 웹 버전에서 F12 (개발자 도구) -> Network 탭 접속

아무 메시지나 보낸 후, messages 항목을 클릭

Request Headers의 Authorization 값을 복사합니다.

코드의 TOKEN 변수에 해당 값을 넣습니다.

3. 실행
Bash


python test.py
⚙️ 설정 커스터마이징
스크립트 상단의 STATUS_TYPES와 STATUS_MESSAGES를 수정하여 자신만의 상태 메시지를 설정하세요. 25초 미만으로 간격을 설정하는 것은 계정 보호를 위해 권장하지 않습니다.

Python


@tasks.loop(seconds=25.0) # 변경 간격 조절
